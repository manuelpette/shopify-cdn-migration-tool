# Migrate Files between Shopify instances

A quick migration tool for Shopify files while we wait for Toby to give us API for files.

### How to install:

1. Download folder from github into your computer.
2. Unzip file into a folder (remember where, you're going to need to load the folder into chrome)
3. Install in chrome going to this url:

```

chrome://extensions

```
4. Click on "Load Unpacked" button
5. Select folder from step 2
6. Operate extension from chrome extensions bar, instructions on how to use are included in the extension popup
7. Enjoy and make the most of your new chrome extension!
